Excel File Upload Guide

Using excel files is a quick and easy way to upload your workouts to the Level Train system. Following this guide will ensure that your workout can be successfully uploaded. If you have any questions, feel free to email {{=site_name}} administration at {{=site_email}}.

General

General

  • Your spreadsheet data MUST be on Speadsheet1 of the excel file.
  • Each section of the workout is separated by a blank line
  • Each section of the workout is imported in sequential order (first section assigned A1, second assigned B1, third C1, etc.)
  • IMPORTANT: The end of the spreadsheet must have a row with a * in the first column. This indicates the end of the workout. See the full example below.

Headers

The headers specify which columns will be read and understood by the system. Headers will be properly be read by the system if they follow these requirements:
  • The headers MUST all appear on the same line.
  • A blank line MUST appear before the headers, unless it is the first line of the spreadsheet.
  • The spelling of any header MUST exactly match one of the following keywords:
    • Upper Push - Exercise category
    • Upper Pull - Exercise category
    • Lower Push - Exercise category
    • Lower Pull - Exercise category
    • Core - Exercise category
    • Weight - If this is set to a number, it will be ignored. Should be empty or set to a word (ex. band, bw, bybar, etc.)
    • Exercise - Name of an exercise; all ascii characters accepted
    • Set 1 - Number of reps for first set of this exercise. (ex. 6, 10R/L, etc.)
    • Set 2 - Number of reps for second set of this exercise
    • Level-Up - Number of reps needed to proceed to next level
    • Variance - Percent of client's body weight for this exercise (ex. Body Weight:150lbs & Variance:0.5 -> Weight:75lbs). Used to calculate the weight for this exercise.
      • You may set this to a specific weight by surrounding with quotes (ex. "10", "12.5")
      • You may also set this to a word (ex. band, bw, bybar, etc.)
  • Headers of other columns that do not match any of these keywords will be ignored.
  • The order of appearance of these headers does not matter.

Exercise Categories

The exercise category is typically the first column of a workout. It determines which muscle group is used during this set of exercises. This column also is used when creating reports of client progress through the {{=site_name}} workout.

These muscle groups are broken up into the following five categories:
  • Upper Push
  • Upper Pull
  • Lower Push
  • Lower Pull
  • Core
Any other input category will be ignored.

More information about categories can be found by viewing a client report.

Main Data

  • Every row following the headers will be read into the database.
  • An empty line MUST NOT appear immediately after a header line.
  • Each row indicates a level in ascending order.
  • The user is able to import as many levels as they would like for a workout section.
  • A blank line signifies the end of a workout section.
  • The first level (first row) will be the initial exercise shown to the client.
  • You can alter the current level by going to the client's workout page and pressing the arrow buttons.

Accessories

On the same line as an exercise for a workout, accessories can be added for pairing exercises together. The headers for these accessories MUST be on the same line as regular exercises, and the following keywords are accepted:
  • Accessory - Name of an exercise; all ascii characters accepted
  • Weight - If this is set to a number, it will be ignored. Should be empty or set to a word (ex. band, bw, bybar, etc.)
  • Set 1 - Number of reps for first set of this exercise. (ex. 6, 10R/L, etc.)
  • Set 2 - Number of reps for second set of this exercise
  • Variance - Percent of client's body weight for this exercise (ex. Body Weight:150lbs & Variance:0.5 -> Weight:75lbs)

Primary

For each week of a workout, a primary exercise can be added for the client to perform at the beginning of the workout. The primary section is similar to the other sections, and the headers of a primary section are as follows:
  • Accessory - Name of an exercise; all ascii characters accepted
  • Weight - If this is set to a number, it will be ignored. Should be empty or set to a word (ex. band, bw, bybar, etc.)
  • Set 1 - Number of reps for first set of this exercise. (ex. 6, 10R/L, etc.)
  • Set 2 - Number of reps for second set of this exercise
  • Variance - Percent of client's body weight for this exercise (ex. Body Weight:150lbs & Variance:0.5 -> Weight:75lbs)
